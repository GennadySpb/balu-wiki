Handling PND Files 
===================

* Run in console with pnd_run 
* Mount PND File
* /usr/pandora/scripts/pnd_run.sh -m -p <pnd_file> 


Installing software 
====================

* opkg install <ipk-file>
* Or use `Milky <http://apps.openpandora.org/cgi-bin/viewapp.pl?/Other/milkyhelper.inf>`_ (pacman for pandora)
